# 🏮 Anke — O Sábio dos Debates Shinobi

**Anke** é um bot de nova geração focado em transformar a forma como sua comunidade discute animes. Desenvolvido para oferecer análises técnicas e profundas, o Anke utiliza Inteligência Artificial (Llama 3.3) para garantir que as discussões no seu servidor subam de nível.

## 🚀 Funcionalidades Principais

* **🧠 /debate**: Inicia uma análise técnica e crítica sobre qualquer tema, personagem ou enredo de animes.
* **💡 /dicas**: Entrega curiosidades raras e fatos obscuros sobre obras japonesas.
* **💬 Chat Inteligente**: O Anke possui memória de contexto! Ao mencioná-lo, ele responde mantendo o fio da conversa, agindo como um membro experiente da comunidade.
* **⚡ Performance**: Respostas rápidas e limpas através de Embeds profissionais.

## 🛠️ Tecnologias Utilizadas

* [Node.js](https://nodejs.org/) - Ambiente de execução.
* [Discord.js](https://discord.js.org/) - Biblioteca para interação com a API do Discord.
* [Groq Cloud SDK](https://console.groq.com/) - Processamento de linguagem natural (IA).
* [Dotenv](https://www.npmjs.com/package/dotenv) - Gerenciamento de variáveis de ambiente e segurança.
